# Como configurar Typescript rápido y fácil

Podríamos ver una a una cada config de Typescript, pero queda fuera del alcance de este artículo. Acá les dejo la [documentación oficial](https://www.typescriptlang.org/tsconfig) por sí sos curioso.

Sin importar si usas, node, next.js, vite, deno, bun o tsx, la forma más rápida de configurar tu proyecto es con este repo.

[TSConfig bases](https://github.com/tsconfig/bases)

Podés instalar cualquiera de los paquetes o como yo, copias y pegas la config y la modificas a gusto.

## Aprendiendo

La parte más interesante del repo, fuera de las configs en sí, es poder abrir cada config y ver que cambia entra cada runtime y bundler, he aprendido mucho haciendo esto y si te gusta Typescript seguramente vos también.
