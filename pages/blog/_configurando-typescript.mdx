# Configurando Typescript

Podríamos ver una a una cada config de Typescript, pero queda fuera del alcance de este articulo. Acá les dejo la [documentación oficial](https://www.typescriptlang.org/tsconfig) por si sos curioso.

Sin importar si usas, node, next.js, vite, deno, bun o tsx, la forma más rápida de configurar tu proyecto es con este repo.

[TSConfig bases](https://github.com/tsconfig/bases)

Podes instalar cualquiera de los paquetes o como yo, copias y pegas la config y la modificas a gusto.

## Aprendiendo

La parte más interesante del repo, fuera de las configs en si, es poder abrir cada config y ver que cambia entra cada runtime y bundler, he aprendido mucho haciendo esto y si te gusta Typescript seguramente vos también.
